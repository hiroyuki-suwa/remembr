# re:membr
A minimal vocabulary & task card app built with Next.js + TypeScript.

## Tech Stack
- Next.js (App Router)
- TypeScript
- Tailwind CSS
- Redux Toolkit

## Local Development
### Installation
```sh
$ yarn
```

### Start the Development Server
```sh
$ yarn dev
```

### Lint Check
```sh
$ yarn lint
```
